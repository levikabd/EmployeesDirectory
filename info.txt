* Приложение запускается из консоли с параметрами. Первый параметр, передаваемый при запуке, выбирает режим работы приложения соответствующий пункту задания. 
* Для ФИО используется английский язык. 
* Приложение  подключается к базе данных Postgresql, используются библиотеки PQXX.
* В разработке применяется объектно-ориентированный подход.
* В качестве среды разработки используется язык программирования С++.

**Режимы работы приложения:**

* 1. Создание таблицы с полями справочника сотрудников, представляющими "Фамилию Имя Отчество", "Дату рождения", "Пол".

**Пример запуска приложения:**
EmployeesDirectory 1

* 2. Создание записи справочника сотрудников.
Для работы с данными создается класс и объект класса с введенными пользователем данными. 
Генерируется строка, создается объект и он отправлятся в базу данных.
У объекта должны быть следующие методы:
	* отправляет объект в базу данных;
	* рассчитывает возраст (полных лет).

**Пример запуска во 2 режиме:**
EmployeesDirectory 2 "Ivanov Petr Sergeevich" 2009-07-12 Male

* 3. Вывод всех строк справочника сотрудников, с уникальным значением ФИО+дата, отсортированный по ФИО. 
Выводится ФИО, дату рождения, пол, кол-во полных лет.

**Пример запуска приложения:**
EmployeesDirectory 3

* 4. Заполняется автоматически 1000000 строк справочника сотрудников. Распределение пола поочередное. Начальная буква ФИО также. 
Дополнительно у класса есть метод, который пакетно отправляет данные в БД, принимая массив объектов.
Заполняется автоматически 100 строк в которых пол мужской и Фамилия начинается с "F".

**Пример запуска приложения:**
EmployeesDirectory 4

* 5. Вывод выборки из таблицы по критерию: пол мужской, Фамилия начинается с "F". Делается замер и вывод времени выполнения.

**Пример запуска приложения:**
EmployeesDirectory 5

* 6. Производится оптимизация базы данных, добавляется индекс к таблице сотрудники для ускорения выполнения пункта 5.
